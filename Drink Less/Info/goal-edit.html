Type: You can set goal for how many alcohol free days you want each week, the maximum number of units you want to drink, the maximum amount you want to spend or the maximum number of calories you want to consume. All relate to a week’s consumption.

Max: Enter a figure for the type of goal you've set. For example: 3 alcohol free days, %@ units, £%@ or %@ calories.

Recurring: Leave this on and the goal will start again each week.
